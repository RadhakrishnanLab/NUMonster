Display tips for switching to RHEL

The display manager used in RH9, XFree86 has been overlooked, or upgraded, depending on how you look at it, for Xorg.

The configuration files for the two are similar, yet different so before you do upgrade RH9 do the following:

  • Make a copy of /etc/X11/XFree86-4.conf on the machine

  • Compare it to the xorg.conf template from Overhauser

  • In the 'Section' called 'Device' in the XFree86-4.conf file, take note of the name of the 'Driver'

If, when upgrading RH9 to RHEL, you have trouble with the display, make sure that the 'Driver' for the 'Device' is the SAME NAME!

If there is no xorg.conf file, you should run 

Xorg
to generate it automatically. However, don't change the generated file unless you have problems.
