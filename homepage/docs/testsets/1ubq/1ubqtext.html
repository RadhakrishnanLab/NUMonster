Overview
  • The point here is to show that Monster can analyse intra-molecular interactions.
Methods
  • In this example the PDB file 1UBQ was loaded via the monster web interface
  • In the parser applet, chain A was copied twice, into chain B and chain C, and then 'ignored'.
  • Chain B had its residue range set to 1-22
  • Chain C had its residue range set to 23-35
  • In the Chain Pairs tab, the one chain pair listed 'BC' was chosen, and the job was started.
  • The results can be viewed through both the 2D bonds viewer, and the 3D webmol viewer(see left).
