Saving and Processing Your Data on Abragam

  1. Recall where you saved your data on Inova; i.e. directory in which you saved the data via. svf('giveanappropriatefilename').
  2. On Abragam, create a folder in your felix980/data directory using mkdir 'fidfoldername' *no capital letters are permitted.
  3. In the folder you just created type sftp ishwar@inova.
  4. When prompted, enter the password of ishwar's account on Inova.
  5. Using cd command, navigate to your data directory (i.e. /space2/home/swanson/data/)
  6. Confirm you have found your data by typing ls which should list files procpar, text, log, and fid
  7. Type command mget * and wait for your data to copy from Inova to Abragam.
  8. When transfer is complete, type exit to return to abragam prompt.
  9. You must now edit a felix macro using the vi editor.
