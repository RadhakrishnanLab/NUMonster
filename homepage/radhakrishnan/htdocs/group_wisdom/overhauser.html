Overhauser Hardware

Win 2K had trouble recognising some of Overhauser's hardware because it uses an Intel Chipset (specifically Chipset 820).

Of note is the Chipset Audio Drivers, downloadable at the previous link.

Also there's:

  • Ethernet Card: Intel(R) PRO/1000 MT

  • Sound: Creative SB Live! Value (There's a CD in the Overhauser pack)

If you install these three set of drives, you should get every device recognised in the device manager.
